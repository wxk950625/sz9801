与 .git同级的是工作区域  也就是使用idea能够打开的.是明文的,通过检出命令.得到的.
本地仓库的位置:  在.git文件夹的下一层.   (仓库中的代码是经过压缩的)
objects目录下是放置代码的位置;


		使用可视化工具创建一个git.  版本库:
						1.创建一个文件夹-->鼠标右键 -->	Git Bash Here--->输入命令   git init  就完成了	
						
操作git:			创建版本库:   1.创建一个文件夹   -->右键-->选择 Git在这里创建版本库
					
					
					添加文件夹:		选中要添加的文件 -->右键  -->TortoiseGit   -->选择添加.   点击确定之后文件会多出一个+号的蓝色标志.
									右键点击Git 提交  填写  备注...确定之后就能添加到版本库浏览器了   (右键 -->TortoiseGit   选择版本库浏览器就能够查看添加的文件了.),提交的代码会保存在objects文件夹里.
					
					
					修改文件:    选中有绿色对勾的文件,双击打开,添加或者修改代码.保存之后,,这个文件的图标会变成红色感叹号.  表示仓库中的代码与我自己的代码不一致,有变更..,加入不想要修改过的文件,可以右键点击TortoiseGit-->还原,
								就能够还原到修改之前的代码了.
				
					
					删除文件:		右键-->TortoiseGit-->删除,点击提交之后,版本库浏览器与这个文件都会删除掉.
									右键-->TortoiseGit-->删除并保留本地副本,提交之后,把这个文件不受版本控制勾子取消,确定之后,版本浏览器中就没有了,但是本地副本还存在.
